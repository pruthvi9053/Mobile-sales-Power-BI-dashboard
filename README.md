# Mobile-sales-Power-BI-dashboard

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze mobile phone sales data.  
The dashboard helps visualize key business metrics such as total sales, transactions, customer ratings, and sales distribution across cities.

## Dashboard Features
- Total Sales KPI
- Average Price KPI
- Total Transactions
- Total Quantity Sold
- Sales by Day Analysis
- City-wise Sales Map
- Payment Method Distribution
- Mobile Model Sales Comparison
- Customer Ratings Visualization

## Tools & Technologies
- Power BI
- Data Visualization
- Data Modeling
- DAX
- Microsoft Bing Maps

## Dashboard Insights
The dashboard allows users to:
- Track overall mobile sales performance
- Identify top-selling mobile brands
- Analyze payment method preferences
- Compare sales performance across cities
- Understand customer satisfaction through ratings

## Dataset
The dataset contains information such as:
- Mobile Model
- Brand
- City
- Payment Method
- Units Sold
- Price Per Unit
- Customer Rating
- Transaction ID
- Date
